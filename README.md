# ARMTox86
This program takes in a string of ARM assembly code, and it converts the code to x86 assembly. So far, we have the mov, add, and push instructions done.

### Instructions Done:  
mov  
add, mul, sub  
push, pop  

### Instructions Needed:  
branching (jumps)  
comparisons (cmp)  
bitwise  
and more...  

### Features In Progress:  
GUI  
file input & output
